RTL

Ab Bootstrap 3.2 beinhaltet das Projekt eine von-rechts-nach-links-Version der Stile. Diese wird von unserem Gruntfile mit der Hilfe von Twitters Projekt CSS Flip generiert.

Verwendung

Bootstrap ist standardmäßig ein von-links-nach-rechts-Projekt. Für von-rechts-nach-links-Projekt musst du deine Sprache festlegen und das Standard-Bootstrap-CSS mit einer RTL-Version ersetzen. Leg zuerst deine Sprache und Schreibrichtung fest:

Alternativ kannst du die komprimierte Version der RTL-Datei verwenden, bootstrap-rtl.min.css.

CSS Flip

CSS Flip ist ein Projekt zur Konvertierung von links-nach-rechts-CSS-Dateien in rechts-nach-links-CSS-Dateien. Wir verwenden es in unserem Gruntfile, um die Generierung von Bootstraps RTL-CSS-Dateien zu automatisieren.
