Markup

Data-Attribute sind essentiell für das Button-Plugin. Sieh dir den Beispiel-Code unten für die verschiedenen Markup-Arten an.

Optionen

Keine

Methoden

$().button('toggle')

Schaltet das "Gedrückt"-Aussehen um. Gibt dem Button den Anschein, dass er aktiviert wurde oder entfernt diesen wieder.

Automatische Umschaltung

Du kannst die automatische Umschaltung bei einem button mit dem Attribut data-toggle aktivieren.

{% highlight html %} {% endhighlight %}

$().button('loading')

Ändert den Status des Buttons zu "ladend". Deaktiviert den Button und ändert den Text zum Laden-Text. Dieser Text sollte am Button-Element mit dem Data-Attribut data-loading-text definiert werden.

{% highlight html %} {% endhighlight %}

Browser-Kompatibilität

Firefox speichert jeglichen besonderen Status eines Formularelements (inkl. Buttons) auch wenn die Seite neu geladen wird. Ein Workaround für dieses Problem ist die Verwendung von autocomplete="off".

$().button('reset')

Setzt den Status des Buttons zurück. Ändert den Text wieder zum Original.

$().button(String)

Setzt den Status des Buttons zurück. Ändert den Text zum angegebenen String.
