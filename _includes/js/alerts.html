Warnhinweise alert.js

Beispiel-Warnungen

Ermögliche mit diesem Plugin das Schließen von allen Warnhinweisen.

Du siehst ein bisschen blass aus! Ist alles in Ordnung?

Oh nein! Es ist ein Fehler aufgetreten!

Verändere dies und das und versuche es dann erneut. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit. Cras mattis consectetur purus sit amet fermentum.

Verwendung

Ermögliche das Schließen eines Warnhinweises über JavaScript:

{% highlight js %}$(".alert").alert(){% endhighlight %}

Markup

Füge einfach data-dismiss="alert" zu deinem Schließen-Button hinzu, um einem Warnhinweis Schließen-Funktionalität hinzuzufügen.

{% highlight html %}{% endhighlight %}

Methoden

$().alert()

Verpackt alle Warnhinweise mit Schließen-Funktionalität. Damit sie beim Schließen langsam ausblenden, gib ihnen vorher die Klassen .fade und .in.

.alert('close')

Schließt einen Warnhinweis.

{% highlight js %}$(".alert").alert('close'){% endhighlight %}

Ereignisse

Bootstraps Warnhinweis-Klassen legen einige Ereignisse offen, mit denen du dich in die Warnhinweise-Funktionen einklinken kannst.

Ereignistyp Beschreibung
close.bs.alert Dieses Ereignis wird sofort ausgelöst, wenn die Methode close aufgerufen wird.
closed.bs.alert Dieses Element wird ausgelöst, wenn der Warnhinweis vollständig geschlossen wurde (wartet auf CSS-Übergänge).
{% highlight js %} $('#mein-alert').on('closed.bs.alert', function () { // etwas ausführen... }) {% endhighlight %}
